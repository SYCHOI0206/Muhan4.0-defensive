SOXL V4 방어형 v11 주문수량 표시 검증

변경 목적
- 매수 주문표의 각 가격 행에 '그 가격에서 실제 주문할 수량'만 표시한다.
- 누적 총수량 표시는 제거한다.

표시 규칙
1. 별지점 매수: 별지점 가격에 주문할 계산수량 표시
2. 평단가 매수: 평단가 가격에 주문할 계산수량 표시
3. 추가 1주 LOC: 각 LOC 가격마다 항상 1주 표시
4. 전반전과 후반전 모두 동일한 표시 원칙 적용
5. 오늘 기록 모달의 LOC 사다리도 각 행 1주로 표시

변경하지 않은 항목
- 매수예산과 가격 사다리 계산
- 자동매매/수동수량 처리
- T 증가 규칙
- 급락 익일 0.25T
- 리버스 25/5 및 버킷 로직
- 실현손익 및 보유평가액 계산

정적 검증
- v40_defensive_patch.js에 '총 ${qty(...)}주' 형태의 LOC 누적수량 표시가 남아 있지 않음
- 메인 주문표 LOC 행: <span>1주</span>
- 오늘 기록 모달 LOC 행: <span class="tag hideQtyTag">1주</span>
- JavaScript 문법 검사 통과
